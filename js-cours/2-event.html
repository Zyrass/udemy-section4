A quoi sa sert ?

Un évènement, est très souvent associé à un clic de souris, également on peut l'utiliser pour stylisé du html avec du CSS. Mais aussi, nous pouvons
caché, affiché des élements, affiché un contenu selon des critères prédéfinis etc...

Evènement mouseenter() & mouseout()

Exemple d'un évènement ".mouseenter()" & ".mouseout()" permettant d'afficher 2 images

Ci-dessous une image d'un logo de final fantasy 15. Si on entre avec la souris dans la div contenant la classe "autreImage" avec la souris,
L'image par défaut changera pour afficher une autre image

Le code html

                <div class="autreImage">
                    <figure>
                        <figcaption>Mon logo FF 15</figcaption>
                        <img src="http://image.jeuxvideo.com/medias-md/146359/1463589413-1698-card.png" alt="logo FF15" width="650" height="417" />
                    </figure>
                </div>

Le code Jquery utilisé

            $(document).ready(function(){
                // Exemple du ".mousseenter()" et du ".mouseout()"
                $("div.autreImage").mouseenter(function(){
                    $('div.autreImage img')
                        .removeAttr("src")
                        .attr("alt", "teste")
                        .attr("src", "http://www.fsmwarden.com/developpez_4/jmduchesne_Individus(IdRel)Equipes_ressources.png");
                    $('div.autreImage figcaption').text("Autre image");
                });

                $("div.autreImage img").mouseout(function(){
                    $('div.autreImage img')
                        .removeAttr("src")
                        .attr("alt", "Mon Logo FF 15")
                        .attr("src", "http://image.jeuxvideo.com/medias-md/146359/1463589413-1698-card.png");
                    $('div.autreImage figcaption').text("Mon logo FF15");
                });
            }); // Fin jquery

Autre évènement ".mouseenter()" mais avec ".mouseleave()"

Autre évènement ".mouseenter()" mais avec ".mouseleave()"

Ici je vais créer 2 block de la même taille. Une fois qu'on entre dans le block bleu, le block rouge grandit.
Quand nous quittons le block bleu, le block rouge redevient normal.

Bleu
Rouge

Code HTML

                <div class="container">
                    <div class="blue">Bleu</div>
                    <div class="red">Rouge</div>
                </div>

Code Jquery

                $(document).ready(function(){

                    // Exemple du ".mouseenter()" et du ".mouseleave()"
                    $("div.container div.red")
                        .text("Yeah je fais 60% au lieu de 30%")
                        .css("width", "60%");
                    });

                    $("div.container div.blue").mouseleave(function(){
                        $("div.container div.red")
                            .text("Rouge")
                            .css("width", "30%");
                    });

                    $("div.container div.red").mouseenter(function(){
                        $("div.container div.blue")
                            .text("Yeah je fais 60% au lieu de 30%")
                            .css("width", "60%");
                    });

                    $("div.container div.red").mouseleave(function(){
                        $("div.container div.blue ")
                            .text("Bleu")
                            .css("width", "30%");
                    });
                }); // Fin jquery
