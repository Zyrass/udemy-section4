2 - Les évènements dans JQuery

A quoi sa sert ?

Un évènement, est très souvent associé à un clic de souris, également on peut l'utiliser pour stylisé du html avec du CSS. Mais aussi, nous pouvons
caché, affiché des élements, affiché un contenu selon des critères prédéfinis etc...

Exemple d'un évènement click permettant de cacher un texte

Exemple d'un évènement click permettant de cacher un texte

Pour notre exemple, je vais créer 2 bouttons permettant de cacher mais également d'afficher un paragraphe.
Au départ le paragraphe est visible.

code jquery

                $(document).ready(function(){
                    $("#cachez").click(function() {
                        $('p q.cacherMoi').hide('slow');
                    });

                    $("#affichez").click(function() {
                        $('p q.cacherMoi').show('slow');
                    });
                });
