Pourquoi JQuery

Pourquoi Jquery

Jquery permet de manipuler les éléments HTML, CSS ou bien, ajouter des animations.. etc...
Nous pouvons aussi gérer la gestion des évènements (le clic, sur un hover etc...). Ainsi on y gagne un temps considérable.
Il faut savoir aussi que quand on code on a souvent des problèmes de compatibilité avec nos navigateurs. En gros un code spécifique
ne sera pas interprété de la même manière sur firefox, chrome ou bien safari etc..
(Souvenez-vous : -webkit- pour chrome ; o- pour opera; ms- pour microsoft etc.. ; )

OU ECRIRE DU CODE JQUERY (a placer dans un fichier script personnel)

OU ECRIRE DU CODE JQUERY (a placer dans un fichier script personnel)

Exemple :

                1 |-> On charge en 1ER jquery
                <script src="https://code.jquery.com/jquery-3.1.1.min.js" ></script>
                

                2 |-> On charge ici notre script Jquery
                <script src="myScript.js" ></script>

2 versions régulièrement disponible

2 versions régulièrement disponible
version Minifié (minified)

Vous trouverez régulièrement des versions minified à disposition du grand public. Elle est compressé, dépourvu des commentaires, des espaces et saut de ligne.
Ainsi il se trouve que le fichier est clairement plis légé et donc plus rapide que sa version non compréssé. Elle est tout le temps utilisé pour la production

version Non-Compressé

Cette version est uniquement utilisé pour le développement. Cette version comprends les espaces, saut de ligne, commentaires etc... En sommes elle est plus simple à relire

UTILISER JQUERY DE PLUSIEURS FACONS

UTILISER JQUERY DE PLUSIEURS FACONS

1 - HTML DIRECTEMENT

A la fin de notre code (avant la balise de fermeture du body) afin qu on laisse le DOM se charger, ouvrons notre balise script soit :

                <script> </script>
Enfin entre ces balises, nous devrons écrire le code jquery avant bien entendu vous aurez ajouté votre script JQUERY

Exemple d'un script jquery

                        code HTML ici...

                        <script src="https://code.jquery.com/jquery-3.1.1.min.js"></script>
                        <script>
                            $(document).ready(function(){
                        
                                // votre code ici...
                        
                            });
                        </script>
                    </body>
                </html>

2 - FICHIER EXTERNE

Même chose en fin du code html (pour rappel, avant </body>), il faut ouvrir une balise script, sauf que là
on y ajoute un attribut src (source) et on ajoute notre fichier qui aura comme extension le .js.
Attention de ne pas oublié le fichier jquery qui pour rappel doit être placé en 1er.

Exemple :

fichier html :
                        code HTML ici...

                        <script src="https://code.jquery.com/jquery-3.1.1.min.js"></script>
                        <script src="myScript.js"></script>
                    </body>
                </html>
myScript.js :
                $(document).ready(function(){
                        
                    // votre code ici...
                        
                });

3 - CDN (Content Délivery Network)

Un CDN est un regroupement de plusieurs version d'un framework hébergé sur un serveur distant.
l'avantage d'un fichier CDN c'est qu'il n'est aucunement nécéssaire d'avoir ces fichiers en local.
l'inconvéniant et oui il y en a un, c'est qu'il faut être connecté sur internet constament pour que notre code fonctionne...
une coupure et c'est le drame.

Exemple :

fichier Jquery 3.X : Minified
                        <script src="https://code.jquery.com/jquery-3.1.1.min.js">
fichier Jquery 3.X : Non-Minified
                        <script src="https://code.jquery.com/jquery-3.1.1.js">
fichier Jquery 2.X : Minified
                        <script src="https://code.jquery.com/jquery-2.2.4.min.js">
fichier Jquery 2.X : Non-Minified
                        <script src="https://code.jquery.com/jquery-2.2.4.js">
fichier Jquery 1.X : Minified
                        <script src="https://code.jquery.com/jquery-1.12.4.min.js">
fichier Jquery 1.X : Non-Minified
                        <script src="https://code.jquery.com/jquery-1.12.4.js">
                    
                $(document).ready(function(){
                        
                    // votre code ici...
                        
                });

Syntaxe de JQUERY

Version classique

Syntaxe de JQUERY CLASSIQUE 
                $(document).ready(function(){
                
                    // votre code ici...
                
                });

document ------------ [Quand notre code dans le body est prêt]
.ready ---------------- [On charge le document. A savoir que .ready est un évènement]
(function(){ }); ----- [Exécute notre code ici]

Version raccourçis

Syntaxe de JQUERY RACCOUCIS 
                $(function(){
                
                    // votre code ici...
                
                });

On exécute un code jquery de la même manière avec cette syntaxe
