Cachez un element et le réafficher

Cachez un element avec ".hide()" et le réafficher avec ".show()"

Nous pouvons cachez un élément avec le ".hide()" et le ré afficher avec ".show()"
Il faut savoir qu'on peut utiliser des attributs comme "slow", "fast" ou bien un nombre en millisecondes (1000) pour 1 sec

Exemple :

Ce text peut être cacher via le bouton ci-dessus

Code html

                <div class="hideShow">
                    <button type="button" id="clickHide">Cachez le paragraphe</button>
                    <p>Ce text peut être cacher via le bouton ci-dessus</p>
                </div>

code jquery

                $(document).ready(function(){
                    // Exemple hide & show
                    var ask = 0;
                    $("#clickHide").click(function(){
                        if (ask == 0) {
                            $("div.hideShow p").hide("slow");
                            $("#clickHide").text("Affiché à nouveau le paragraphe");
                            ask = 1;
                        } else {
                            $("div.hideShow p").show("slow");
                            $("#clickHide").text("Cacher de nouveau le paragraphe");
                            ask = 0;
                        }
                    });
                }); // Fin jquery

Cachez un element avec ".fadeOut()" et le réafficher avec ".fadeIn()"

Nous pouvons cachez un élément avec le ".fadeOut()" et le ré afficher avec ".fadeIn()"
Il faut savoir qu'on peut utiliser des attributs comme "slow", "fast" ou bien un nombre en millisecondes (1000) pour 1 sec

Exemple :

Ce text peut être cacher via le bouton ci-dessus en fondu

Code html

                <div class="fadeInShow">
                    <button type="button" id="clickFadeOut">Cachez le paragraphe</button>
                    <p>Ce text peut être cacher via le bouton ci-dessus en fondu</p>
                </div>

code jquery

                $(document).ready(function(){
                    // Exemple fadeOut & fadeIn
                    var askTwo = 0;
                    $("#clickFadeOut").click(function(){

                            askTwo = 1;
                        } else {
                            $("div.fadeInShow p").fadeIn("slow");
                            $("#clickFadeOut").text("Cacher de nouveau le paragraphe");
                            askTwo = 0;

                    });
                }); // Fin jquery

Conception d'un callBack

Conception d'un callBack

un callBack permet de spécifier des effets une fois qu'un certain effet spécifique c'est terminé.

Exemple :
4 paragraphes déjà utilisé précédemment. Quand je clic sur le paragraphe 1 en cascade je cache le p2 p3 p4.
J'ajoute également un bouton permettant en cascade de réafficher les éléments qui ont été caché en fondu

Je suis une paragraphe avec pour numéro le 1

Je suis une paragraphe avec pour numéro le 2

Je suis une paragraphe avec pour numéro le 3

Je suis une paragraphe avec pour numéro le 4

code html

                <div>
                    <p class="p1">Je suis une paragraphe avec pour numéro le 1</p>
                    <p class="p2">Je suis une paragraphe avec pour numéro le 2</p>
                    <p class="p3">Je suis une paragraphe avec pour numéro le 3</p>
                    <p class="p4">Je suis une paragraphe avec pour numéro le 4</p>
                </div>
                <button type="button" id="callBack">Réinitialisation l'exemple d'un callBack</button>

code jquery

                $(document).ready(function(){
                    // Exemple d'un callBack
                    $("div#onTest p.p1").click(function(){
                        $("div#onTest p.p2").hide(2000, function(){
                            $("div#onTest p.p3").hide(2000, function(){
                                $("div#onTest p.p4").hide(2000);
                            });
                        });
                    });
                        
                    $("#callBack").click(function(){
                        $("div#onTest p.p2").fadeIn(2000, function(){
                            $("div#onTest p.p3").fadeIn(2000, function(){
                                $("div#onTest p.p4").fadeIn(2000);
                            });
                        });
                    });
                });

Effet .fadeTo(arg1, arg2)

Effet .fadeTo(arg1, arg2)

Parfois vous voudrez créer un effet pour spécifier une certaine opacité. Avec .fadeTo c'est possible !

syntaxe du fadeTo

                $("h1").fadeTo( arg1, arg2);

                arg1 correspond au délais en milliseconde.
                arg2 correspond lui à l'opacité qui pour rappel va de 0 à 1.

Exemple d'un paragraphe qui une fois cliqué dessus devient partiellement transparent. Au bout de 5sec il retrouve sa forme d'origine

Ceci est un paragraphe qui devient vert avec une opacité de 1 si on sort avec la souris. Mais quand on clic dessus il devient rouge
avec une opacité de 0.5

code html

                    <div class="testFadeTo">
                        <p>Ceci est un paragraphe qui devient vert avec une opacité de 1 si on sort avec la souris.

                        Mais quand on clic dessus il devient rouge avec une opacité de 0.5</p>
                    </div>

code jquery

                    $(document).ready(function() {
                        // Exemple d'un .fadeTo sous forme d'objet
                        $(".testFadeTo").on({
                            click: function(){
                                $(".testFadeTo p")
                                    .fadeTo(1000, .5)
                                    .css("background-color", "#800000")
                                    .css("color", "#fff");
                            },
                            mouseleave: function(){
                                $(".testFadeTo p")
                                    .fadeTo(1000, 1)
                                    .css("background-color", "#008000")
                                    .css("color", "#fff");
                            }
                        });
                    });
