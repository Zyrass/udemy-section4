1 - Les selecteurs "Selectors"

Selectionner un élément avec le $

Selectionner un élément avec le $

$ permet de désigner un élément.
On ouvre obligatoirement des parenthèses et des guillemets simple ( ' ' ) ou double ( " " ) soit : 

                $ ( " element_html " )

Exemple : { Selection d'une balise h5 }

                $("h5")

Selectionner une classe

Selectionner une classe

$ permet toujours de désigner un élément.
Donc comme auparavant, on ouvre obligatoirement des parenthèses et des guillemets simple ( ' ' ) ou double ( " " ). En revanche, comme en CSS pour sélectionner une classe,
on doit y ajouter un POINT suivit du mot qu'on aura définit soit : 

                $ ( " .motX " )

Exemple : { Selection d'une classe avec comme mot "essai" }

                $(".essai")

Selectionner un identifiant

Selectionner un identifiant

C'est exactement comme pour une classe mais avec le selecteur # (aussi appelé sharp)

                $ ( " #motY " )

Exemple : { Selection d'un identifiant (ID) avec comme mot "teste" }

                $("#teste")

Selectionner un element avec les pseudos selecteur

Selectionner un element avec les pseudos selecteur

en CSS, nous pouvons utiliser les pseudos selecteur comme par exemple :

  • :nth-child(x)
  • :first-child
  • :last-child
  • etc...
En jQuery c'est la même chose

                $ ( " p:nth-child(odd) " )

Exemple : { Selection du 3ème paragraphe }

                $("p:nth-child(3)")
