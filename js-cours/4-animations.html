4 - Les animations dans JQuery

Les animations

Explication sur les animations

Les animations sont courament utilisé dans jquery. La syntaxe ressemble fortement à l'évènement .on avec plusieurs arguments à renseigner.

Syntaxe de .animate()

                $("selecteur").animate( arg1, arg2, arg3);

                arg1 Correspond à un objet contenant toutes les propriétés qu'on va appliqué.
                arg2 Correspond au délais qu'on souhaite pour notre animation (optionnel).
                arg3 Correspond à un callBack, c'est à dire un code à exécuté en fin d'annimation. (optionnel).

Exemple : Conception ci dessous d'un bouton permettant de lancer une animation.
Cette animation permettra de changer un carré en rectangle avec un délais de 1seconde.
En tant qu'objet il y aura 2 propriétés qui changeront (l'opacité et la largeur de l'élément).
De plus nous y ajoutons un callBack permettant de revenir à sa position d'origine une fois l'animation terminé.

code html

                <div>
                    <button type="button" id="launchAnimate">Cliquez ici pour lancer l'animation</button>
                    <div class="formAnimate"></div>
                </div>

code jquery

/!\ important /!\
width: "+=300px" veut dire qu'on ne sait pas combien fait notre carré au départ ainsi donc on ajoute à notre width d'origine +300 px

                $(document).ready(function(){
                    // Exemple d'une animation
                    $("#launchAnimate").click(function(){
                        $(".formAnimate").animate({
                            width: "+=300px",
                            opacity: .5
                        }, 1000, function(){
                            $(".formAnimate").animate({
                                width: "100px",
                                opacity: 1
                            }, 1000);
                        });            
                    });
                });

fonction .stop() sur une animation

fonction .stop() sur une animation

la fonction stop() permet comme son nom l'indique de stopper une animation
si bien entendu le code correspondant à bien été attribué à un bouton spécifque pour stopper une animation.
La fonction .stop() peut prendre deux arguments qui par défaut sont à "false". Si on applique pour le premier "true",
celà permet de stopper TOUTES les animations en cours alors que sans cette arguement
si il existe plusieurs animations qui s'éxécute à la suite, dans ce cas, quand on cliquera sur stop,
l'animation suivante s'exécutera tout de même. Le second argument permet de sauter l'animation et d'allez jusqu'à la fin sans la joué

syntaxe de la function .stop()

                $("selector").stop(arg1, arg2);

                arg1 Correspond à un arrêt total de toutes les annimations 
                     si cette dernière est passé sur "true". ("fasle" par défaut).
                     
                arg2 Correspond ici à une annulation de l'animation en cours en affichant
                     directement le résultat attendu avec "true". ("fasle" par défaut).

Exemple :

code html

                <div>
                    <button type="button" id="startAnimate">Start</button>
                    <button type="button" id="stopAnimate">Stop</button>
                    <button type="button" id="initAnimate">Ré-initialisation</button>
                    <div id="divAnimate"></div>
                </div>

code jquery

                // Exemple du stop sur une animation
                $("#startAnimate").click(function(){
                    $("#divAnimate").animate({
                        width: "+=200px",
                        height: "+=200px",
                        'border-radius': '50%',
                    }, 1000)
                });

                $("#stopAnimate").click(function(){
                    $("#divAnimate").stop(true);
                });

                $("#initAnimate").click(function(){
                    $("#divAnimate").animate({
                        width: "50px",
                        height: "50px",
                        'border-radius': '0%',
                        'background-color': '#ff80ff'
                    }, 1000)
                });
